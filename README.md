# weiqi tui

## How to:
Building:
```
stack build
stack exec weiqi-exe
```

Enter your piece placement in the form of:
```
<x-coordinate> <y-coordinate> (Black|White)
```

*Note: coordinates are zero-indexed*

## Features:
- Seed removal
- 9x9 Board
